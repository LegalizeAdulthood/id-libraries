	This directory contains a collection of files that can be used
to create and display a variety of L-Systems using the Fractint program
L-System parser. To use them, copy the file(s) to your Fractint directory
and then execute the Fractint program. Select the fractal type "lsystem"
and then use the F6 function key to enable an alternate l-system file
selection. If you want to contribute L-Systems of your own to this directory, 
please send your submissions to:

	William McWorter 
	mcworter@midohio.net
